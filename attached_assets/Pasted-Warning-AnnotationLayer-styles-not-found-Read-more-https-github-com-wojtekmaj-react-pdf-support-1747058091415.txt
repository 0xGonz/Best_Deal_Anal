Those new console messages are harmless warnings from react‑pdf.
They appear because the library tries to render two optional overlay layers and can’t find the matching CSS files:

Text Layer – positions selectable text so users can copy/search.

Annotation Layer – draws links, form fields, highlights, etc.

Without the styles the PDF still renders, but:

Text selection may look mis‑aligned.

Link hit‑boxes may be off.

You get the console noise every time a page mounts.

You have three tidy ways to address it.

Option A – Import the official styles (recommended)
Add these two one‑liners once near the top of your main bundle (src/main.tsx or App.tsx):

ts
Copy
Edit
import 'react-pdf/dist/esm/Page/TextLayer.css';
import 'react-pdf/dist/esm/Page/AnnotationLayer.css';
Vite will bundle them; the warnings disappear and the overlays render correctly.

If you use Tailwind or custom resets
Those CSS files set some absolute positioning rules. They rarely clash, but if you see any oddities, add this after your Tailwind base import so its utilities win:

css
Copy
Edit
@tailwind base;
@tailwind components;
@tailwind utilities;

@import 'react-pdf/dist/esm/Page/TextLayer.css';
@import 'react-pdf/dist/esm/Page/AnnotationLayer.css';
Option B – Ship your own minimalist styles
If you want full control (or a smaller bundle), copy the bits you need:

css
Copy
Edit
/* pdf-text-layer.css */
.react-pdf__Page__textContent {
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  color: transparent;         /* keeps selectable */
  user-select: text;
  cursor: text;
}

/* pdf-annotation-layer.css */
.react-pdf__Page__annotations {
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  pointer-events: auto;
}
Import those instead of the library defaults.

Option C – Suppress the layers entirely
If you don’t need selectable text or links (e.g. read‑only pitch decks), set props on <Page>:

tsx
Copy
Edit
<Page
  renderTextLayer={false}
  renderAnnotationLayer={false}
/>
The warnings vanish because react‑pdf never mounts those layers.

Fastest fix: import 'react-pdf/dist/esm/Page/TextLayer.css'; import 'react-pdf/dist/esm/Page/AnnotationLayer.css';
